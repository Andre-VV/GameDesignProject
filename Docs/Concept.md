# Game Concept:

A game where the player has to escape/survive from zombies attacking them. Top down camera.

# User Stories:

|Feature|Story|
|:---------|:-----|
|Player Movement|As a player I want to move my character to avoid enemies|
|Enemy AI|As a player I want the enemys to chase me|
|HUD|As a player I want to see my health and ammon count on screen|
|Enemy attack|As a player I want the enemys to melee attack me if they get too close|
|Game Over|As a Player I want the game to end if my health reaches zero|
|Pickups|As a player I want to be able to pick up items to heal me or get new weapons|
|Level complete|As a player I want to win by reaching the end of the level|
